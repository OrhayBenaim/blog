## Menu Access (Alt + key opens menu)

Use Alt + / (Tool Finder) to search for any menu item by name. This is the most reliable way to access menu items via keyboard.

| Menu | Key |
|------|-----|
| File | Alt + f |
| Edit | Alt + e |
| View | Alt + v |
| Insert | Alt + i |
| Format | Alt + o |
| Slide | Alt + s |
| Arrange | Alt + r |
| Tools | Alt + t |
| Help | Alt + h |

## Inserting Elements via Menu

Since there are no direct keyboard shortcuts for inserting elements like text boxes, images, shapes, and tables, use the Tool Finder (Alt + /) and type the element name:

- Text box: Alt + / then type "text box"
- Image: Alt + / then type "image"
- Shape: Alt + / then type "shape"
- Table: Alt + / then type "table"
- Chart: Alt + / then type "chart"
- Video: Alt + / then type "video"
- Word art: Alt + / then type "word art"
- Line: Alt + / then type "line"
