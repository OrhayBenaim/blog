# Blog Post Design: From Idea to Strategy in Minutes

## Meta
- **Type:** Blog post (Markdown)
- **Target read time:** 5-7 minutes
- **Audience:** Mix of technical and non-technical readers
- **Tone:** Human, casual, first-person. No em dashes, no AI-speak. Written from real experience.
- **Post location:** `src/content/blog/from-idea-to-strategy-in-minutes.md`

## Structure

### Section 1: Title & Opening
- **Title:** "From Idea to Strategy in Minutes"
- **Opening (2-3 paragraphs):** Set the scene casually. Describe the workflow at a high level: talk to one AI to refine your thinking, then it routes work to a specialist agent. No hype, no buzzwords. Just "here's what I do and it works." Mention this is the first post in a series going deeper on each tool.

### Section 2: The Setup
- Short section (3-4 paragraphs) explaining the two-layer concept:
  1. **Brainstorming layer:** Start every task by talking through the idea with Claude Code using the Superpowers brainstorming skill. It asks questions, narrows things down, picks an approach.
  2. **Specialist layer:** Once the idea is shaped, hand off to a specialist agent. In this example, the TikTok Strategist. Could be Content Creator, SEO Specialist, Brand Guardian, etc.
- Include install command for Superpowers: `claude plugin add superpowers`
- Mention specialist agents are configurable agent types.
- Link to source: [Superpowers plugin on GitHub](https://github.com/anthropics/claude-plugins-official) (or marketplace link)

### Section 3: The Example Walkthrough (main content)
Walk through a real scenario end to end:
1. **The prompt:** Show the actual prompt typed (e.g., "I need TikTok content for a fitness brand targeting young adults")
2. **Brainstorming phase:** Show 2-3 back-and-forth exchanges as quoted prompt blocks. The skill asking questions, user answering, idea narrowing down.
3. **The handoff:** Show the moment brainstorming completes and the TikTok Strategist agent is invoked. Show the command/prompt that triggers this.
4. **The output:** Show what the TikTok Strategist produced (content ideas, hooks, posting strategy). Include images or video screenshots of final output.
5. **Your take:** Short honest paragraph. What was useful, what you'd tweak, how it compared to doing it manually.

This section is visual-heavy: prompt screenshots, agent output, TikTok content mockups.

### Section 4: The Bigger Picture & Series Teaser
- Short closing (2-3 paragraphs). Zoom out: "TikTok was just one example. Same flow works for different things."
- Tease upcoming posts with one-liners:
  - Building a landing page from a Figma design
  - Generating images and videos with AI (Nano Banana)
  - Writing a full content strategy across platforms
  - Optimizing an app store listing
- End with install commands/links for people who want to try it themselves.

## Tools & Links to Include

- **TikTok Strategist agent:** Include install/config instructions or download link
- **Other agents mentioned (teasers only):** Content Creator, SEO Specialist, Brand Guardian, Nano Banana, Figma integration

## Writing Guidelines
- No em dashes
- No AI-sounding phrases ("revolutionary", "game-changer", "leverage", "delve")
- First person, conversational
- Show real prompts and real outputs
- Include downloadable links or install commands for every tool/skill/agent mentioned
- Images and videos where possible
